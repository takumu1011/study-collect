transformの使い方

※◎transform:translateX(200px) translateY(200px);
✖transform:translateX(200px);
transform:translateY(200px);　←こっちに上書きされる

translate

※自分を基準として動く

※センター寄せ：absolute left:50%; translateX(-50%)


      .translate_parent:hover > .translate{
        transform:translate(300px,20px);
      }

translate(50%)とleft:50%の違い

translate50%


      .wrap:hover>.translate50{
        transform:translateX(50%);
      }

left50%


      .wrap:hover>.left50{
        position:absolute;
        left:50%;
      }

center配置


      .wrap:hover>.center{
        position:absolute;
        bottom:0;
        left:50%;
        transform:translateX(-50%);
      }
