absolute

  • 直近の親要素(static以外)が基準となる。ない場合body要素が基準
  • 包含ブロックの高さがなくなる
  • 他の要素のmarginと相殺されない

高さがなくなった時の高さの出し方
・height
・padding-top(ブラウザの横幅を基準とした可変性)
・vh(ブラウザの縦幅を基準とした可変性)
・Js
