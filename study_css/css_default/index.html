CSS初期値

  • Z-index: auto

    親要素と同じ階層になります。

  • background-repeat: repeat

    縦横に背景画像を繰り返して表示します。

  • background-color: transparent

    背景色を透明にします。

  • background-size: auto auto

    自動的に算出されます。

  • background-position: 0% 0%

    自動的に算出されます。

  • top: auto/bottom: auto

    自動的に調整されます。

  • margin: 0

  • padding: 0

  • list-tyle-type: disc

  • writing-mode: horizontal-tb

  • box-sizing: content-box
