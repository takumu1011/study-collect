animation

animation解説

easing

transitionとの違い

アニメーションに指定できないCSSプロパティもある(例：display:none)

二工程以上のアニメーション時に利用

ループが可能

  • animation-name
  • animation-duration
  • animation-delay
  • animation-timing-function
  • animation-iteration-count：アニメーションのループ回数(数字,infinite)
  • animation-direction：アニメーションの再生方向(reverse,alternate,alternate-reverse)
  • animation-fill-mode：指定したプロパティを開始前・後に適用(forwards,backwards,both)
  • animation-play-state：再生中か一時停止にするか(running,paused)

    @keyframes rotation{
      0%{
        transform:scale(0.3);
      }
      50%{
        transform:scale(0.6) rotate(-45deg);
        background:red;
      }
      100%{
        transform:scale(1) rotate(180deg);
        background:blue;
      }
    }
  

    @keyframes fadeIn{
      from{
        opacity:0;
      }
      to{
        opacity:1;
      }
    }

通常


      .wrap_first:hover>.box{
        animation:rotation 2s;
      }

rotation


      .wrap_second:hover>.box{
        animation:rotation 2s 0s ease-in 3 alternate none running;
      }

fadeIn


      .wrap_third:hover>.box{
        animation:fadeIn 3s 0s ease infinite reverse forwards running;
      }
