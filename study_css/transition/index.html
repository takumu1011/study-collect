animationとの違い

easing

  • transition-property:指定できないCSSプロパティもある(例：display:none)
  • transition-duration
  • transition-delay
  • transition-timing-function

※ショートハンドの場合：durationを先に書く

一工程のアニメーション時に利用

ループがない

戻るアニメーションが自動でつく

トリガーが必要(hoverなど)

通常


      .wrap_second>.box{
        transition:all 3s;
      }
    

      .wrap_third>.box{
        transition:all 3s 2s;
      }

linear:一定に変化


      .wrap_fourth>.box{
        transition:all 3s 0s linear;
      }

ease-in:開始ゆっくり


      .wrap_fifth>.box{
        transition:all 3s 0s ease-in;
      }

steps(数字,start or end)


      .wrap_sixth>.box{
        transition:all 3s 0s steps(6,end);
      }

ease-out:終了ゆっくり


      .wrap_seventh>.box{
        transition:all 3s 0s ease-out;
      }

ease-in-out:開始と終了ゆっくり


      .wrap_eigth>.box{
        transition:all 3s 0s ease-in-out;
      }

opacityのみtransition


      .wrap_ninth>.box{
        transition:opacity 3s 0s ;
      }

どう動くか


      .box-special{
        width:200px;
        height:200px;
        background:#000;
        transition:background 5s ease 1s; ←hoverを外した時に発動(ページロード時効果なし)
      }
      .box-special:hover{
        background:#00ffff;
        transition:background 2s ease 0s; ←hoverした時に発動
      }
