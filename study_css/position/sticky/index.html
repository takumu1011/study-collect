sticky

  • relativeから特定の位置に達した時fixedに変わるイメージ
  • 親や先祖要素にoverflow:hidden,scroll,autoが使われているとうまく動作しない。
  • IEに対応していないためjsで対応

    .block-one{
      background:red;
      z-index:100;
    }
  

    .block-two{
      background:skyblue;
      z-index:200;
    }
  

    .block-three{
      background:yellow;
      z-index:300;
    }
  

    .block-foour{
      background:orange;
      z-index:400;
    }
  

    .block-five{
      background:green;
      z-index:500;
    }
  

    .block{
      width:100%;
      height:100vh;
      position:sticky;
      top:0;
    }
