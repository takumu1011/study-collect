z-index

  • 数字が大きいほど上になる。
  • 通常は上に書かれたものが奥、下になるほど手前に重なっていく。
  • position:staticには使えない。
  • 初期値はauto(親要素と同じ階層に表示される。)
  • z-indexを指定しなくてもstatic要素よりrelative要素が上になる。
